# 🏘️ Brookhaven RP

A browser-based Brookhaven (Roblox) inspired roleplay game. Built with vanilla HTML5 Canvas — no dependencies, works anywhere.

**[▶️ Play Now](https://wilson6040.github.io/brookhaven-rp)**

## Controls

| Key | Action |
|-----|--------|
| `WASD` / Arrow Keys | Move / Drive |
| `E` | Enter/exit car or building |
| `T` | Open chat (type message, press Enter) |
| `C` | Open character customisation |
| `ESC` | Close panels |

## Features

- 🗺️ Open world town with 9 locations: House, School, Hospital, Police Station, Prison, Pizza Place, Bank, Mansion, Car Dealership
- 🚗 4 driveable cars — walk up and press E to get in
- 🏠 Enter any building and explore furnished interiors
- 🎨 Customise your character: skin, shirt and hair colour
- 💬 In-game chat with speech bubbles
- 🗺️ Live minimap in the corner
- 🌙 Day/night cycle (full cycle every 10 minutes)
- 💰 Earn $50 every minute — start with $500

## Tech

Single `index.html` file, zero dependencies, works on GitHub Pages or locally via file://.
